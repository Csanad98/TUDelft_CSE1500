Exercise 01

What is the value of r in the following examples?

Example 1 (hint: unary plus and addition)

var a = "12";
var b = 2;
var r = a+b;
  1. 122
  2. "122"
  3. 14

Example 2

var a = "12";
var b = 2;
var r = a-b;
  1. "1"
  2. "10"
  3. 10

Example 3 (hint: increment and decrement)

var a = 3;
var b = a++;
var c = ++a;
var r = a+b-c;
  1. 3
  2. 4
  3. 8

Example 4 (hint: equality operators)

var a = "3";
var b = 3;
var r = a == b;
  1. true
  2. false

Example 5

var a = "3";
var b = 3;
var r = a === b;
  1. true
  2. false
